This is a proof of concept for a 'distributed app' to share information with permanent storage and open access. It consists of a smart contract to store data on the ethereum blockchain and a web interface to interact with it.

The contract address is 0x8b963ede1E20d93eC2d73afaB54368F07DE0d382

The source code for the smart contract and the web interface is available on github

This CivicTechYYC blockchain experiment is based on and owes a debt of gratitude to the EtherShare project
